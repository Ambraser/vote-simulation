## New generative models

### R-based generators — Dirichlet Diffusion Dynamics (DDD)

Six new generators backed by an embedded R script. They produce utility matrices using Beta-marginal Dirichlet diffusion on an opinion simplex. Requires R with the `MASS` and `randcorr` packages.

| Code | Description |
|---|---|
| `DDD_BETA_0-05` | Beta marginals with alpha-beta in [0, 0.05] |
| `DDD_BETA_0-5` | Beta marginals with alpha-beta in [0, 0.5] |
| `DDD_BETA_1` | Beta marginals with alpha-beta in [0, 1.0] |
| `DDD_BETA_2` | Beta marginals with alpha-beta in [0, 2.0] |
| `DDD_BETA_5` | Beta marginals with alpha-beta in [0, 5.0] |
| `DDD_BETA_POLAR` | Polarised Beta (alpha = beta in [0, 0.5]) |
